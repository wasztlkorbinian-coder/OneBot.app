Features

Alles was dein
Server braucht.

Von Moderation bis Community-Management — OneBot hat jedes Feature das ein moderner Discord Server benötigt.

🎫
Ticket System
Professionelles Support-Ticket System mit Kategorien, Staff-Rollen und automatischen Logs.
/ticket-setup
🛡️
Moderation
Ban, Kick, Mute mit DM-Benachrichtigung, Mod-Log und ephemeral Antworten. Alles protokolliert.
/ban /kick /mute
📝
Bewerbungssystem
Erstelle mehrere Bewerbungen mit eigenen Fragen und automatischer Rollenvergabe bei Annahme.
/application setup
🔔
Social Notifications
Automatische Benachrichtigungen für YouTube, Instagram und TikTok Posts. Nie wieder manuell ankündigen.
/notifications add
🎭
Reaction Roles
Interaktive Panels wo Mitglieder sich selbst Rollen geben können — mit optionaler Remove-Funktion.
/reactionrole create
🎖️
Join Roles
Gib neuen Mitgliedern automatisch Rollen wenn sie dem Server beitreten.
/joinrole add
👋
Welcome Messages
Begrüße neue Mitglieder mit personalisierten Nachrichten, Titel, Farben und Banner-Bild.
/welcome-message setup
🤖
Automod
Automatische Erkennung von Schimpfwörtern, Spam, Invite-Links und Custom-Wörtern.
/automod setup
🎤
Voice Support
Wenn ein User den Support-Channel betritt, werden alle Staff Member automatisch gepingt.
/support-setup

Setup

In 3 Schritten
startklar.

OneBot ist in wenigen Minuten einsatzbereit — kein kompliziertes Setup nötig.

1
Bot einladen
Klicke auf "Bot einladen" und wähle deinen Discord Server aus. Bestätige die Berechtigungen.
2
Commands nutzen
Starte mit /ticket-setup, /automod setup oder /welcome-message setup um Features zu aktivieren.
3
Fertig! 🎉
Dein Server ist vollständig ausgestattet. Alle Features laufen automatisch im Hintergrund.
